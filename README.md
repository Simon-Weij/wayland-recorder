# Wayland recorder

A simple cross-compositor way to record/clip your screen

requirements: <br>

- An wayland compositor e.g mutter
- Pipewire
- XDG desktop portal

## Working desktops

Currently only gnome/mutter is confirmed to work, feel free to test this code on other desktops and let me know.

## Packaging

Currently this project is only packaged for nix, feel free to open a PR to package this code for other distros/package managers
